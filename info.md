# Joyonway P23B32 Spa

Integration locale RS485 via pont USR-W610 pour la centrale Joyonway P23B32.

Expose les commandes confirmees (lumiere, pompes, bulleur, filtration, all off)
comme boutons Home Assistant.

Configuration : IP et port du USR-W610 via l'interface (config flow).

Voir le README pour la liste complete des fonctionnalites et la roadmap.
